Ce document décrit la structure du Google Sheet et les conventions à respecter pour que l'application fonctionne correctement.

⚙️ Onglet Config

C'est l'onglet de configuration globale. Il contient les paramètres de l'admin et le planning de chaque classe.

Ligne 1 — Mot de passe admin

Col ACol B
ADMINmot_de_passe_admin

Le mot de passe admin permet de se connecter depuis n'importe quel compte élève. Il donne accès à toutes les pages sans restriction et bypass le verrou des maxis.

Lignes suivantes — Une classe = deux lignes

Ligne 1 (paramètres de la classe) :

Col ACol BCol CCol DCol ECol F…
Nom de la classeAccès Appli ✓/☐Horaire débutHoraire finDate séance 1Date séance 2…

Ligne 2 (statut des séances) :

Col ACol BCol CCol DCol ECol F…
Label libre (ex: ton nom)videvidevideok / videok / vide…

Règles importantes

  • Accès Appli : case à cocher. ✓ (TRUE) = accès autorisé. ☐ (FALSE) = app bloquée pour les élèves. L'admin garde toujours l'accès.
  • Horaires : format 10h05 ou 10:05. Définissent la plage horaire pendant laquelle une séance est active.
  • Dates : format jj/mm/aa.
  • Statut "ok" : écrire exactement ok (minuscules) sous une date pour que l'app la reconnaisse comme une séance valide. ⚠️ Sans "ok", même si la date et l'heure correspondent, la séance n'est pas active.
  • Séance active = date du jour + statut "ok" + heure actuelle dans la plage horaire.
  • Numéro de séance = nombre de dates avec statut "ok" passées ou en cours (S1, S2, S3…). S1 et S2 = recherche de maxi (non comptées dans le graphique de progression).

Exemple

Ligne 1 :  ADMIN      |  1234

Ligne 3 :  2BPRO  |  ✓  |  10h05  |  11h40  |  10/03/26  |  17/03/26  |  24/03/26  |  31/03/26
Ligne 4 :  Cuvelier   |     |         |         |  ok        |  ok        |  ok        |  (vide)

→ 3 séances validées (S1, S2, S3). La séance S4 du 31/03/26 sera active uniquement si tu écris "ok" et que l'heure correspond.

Valeurs spéciales dans la colonne maxi

  • B : atelier bloqué — n'apparaît pas dans Maxis ni en Séances (machine en panne, indisponible)
  • 0 : valide uniquement pour Banc à Lombaires, Abdo Sol et Gainage sol
  • Vide : maxi non renseigné → l'atelier n'apparaît pas en Séances

📊 Onglets [Classe]_Historique

Un onglet par classe, créé manuellement. Stocke l'historique des séances pour le graphique de progression.

Structure

IndexContenu
0Nom
1Prénom
2Réservé
3S1 — JSON de séance
4S1 — Moyenne ressentis
5S2 — JSON de séance
6S2 — Moyenne ressentis
…S{n} JSON = index 3 + (n-1)×2, moyenne = index suivant

Valeurs spéciales dans les colonnes JSON

  • A : élève absent — affiché comme absence dans le graphique
  • D : élève dispensé — affiché comme dispense dans le graphique
  • Vide : séance non effectuée
Le graphique de progression commence à S3 (S1 et S2 sont des séances de recherche de maxi). La moyenne exclut les ressentis E et les ateliers spéciaux.

✅ Checklist — Ajouter une nouvelle classe

  1. Créer un onglet dans le GSheet avec le nom exact de la classe (ex : 2BPRO)
  2. Ajouter la ligne d'en-tête des colonnes (voir structure ci-dessus)
  3. Saisir les élèves (nom, prénom, classe) — le mot de passe sera défini à la première connexion
  4. Créer l'onglet 2BPRO_Historique avec la même liste d'élèves (colonnes A et B)
  5. Dans l'onglet Config, ajouter les deux lignes de la classe (paramètres + statuts)
  6. Cocher "Accès Appli" pour autoriser l'accès
  7. Ajouter les dates des séances et écrire ok au fur et à mesure

✅ Checklist — Démarrer une séance

  1. Dans l'onglet Config, écrire ok sous la date du jour dans la ligne de statut
  2. Vérifier que les horaires correspondent bien au créneau
  3. Vérifier que "Accès Appli" est coché
  4. Les élèves peuvent se connecter — l'app passe en mode édition automatiquement

✅ Checklist — Période d'évaluation théorique

  1. Dans l'onglet Config, décocher "Accès Appli" pour la classe concernée
  2. Les élèves voient un écran "Application indisponible pendant la période d'évaluation théorique"
  3. L'admin garde l'accès normalement
  4. Recocher la case quand la période d'évaluation est terminée
